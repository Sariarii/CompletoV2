## Présentation

Ce projet consiste à développer une application web permettant à **Lab42c** de suivre les coûts des infrastructures AWS de ses clients, tout en offrant à ces derniers un tableau de bord leur permettant de consulter leurs estimations de facturation. La solution est conçue pour être déployée sur AWS et répondre aux besoins des équipes Lab42c ainsi qu'à ceux de leurs clients.
